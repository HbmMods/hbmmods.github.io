Compatibility Bulletin

This list contains recent changes that need to be accounted for when updating, as well as removed features.

X5628

Huge breaking changes for RBMKS
528 mode now has mandatory reasim control rods which work like standard control rods but require power when moving.
RBMK absorbers now heat up when exposed to neutrons, making them more useful than the arguable cheaper concrete approach. 20 flux equals and extra 1°C.
RBMK passive cooling is now different, there are two values, passive cooling for center columns and passive cooling for edge columns. Columns with four neighbors will cool by 0.1°C/t while columns with no neighbors will cool by 2.5°C/t, values in between are smoothly interpolated. This replaces the old system where columns always cool by 1°C/t.
RBMK coolers no longer inhale cryogel, instead they use cold PFM and output regular PFM, allowing the coolant to be reused with compressors. Coolers also act in a 5x5 area, where all columns will be cooled by 200°C/t, down to a minimum of 20°C. This makes the affected rods unsuitable for power production, but it allows for very rapid shutting down of a hot reactor.
RBMK reasim fuel channels no longer fire neutron streams in six fully randomized directions, but rather in eight which are evenly spaced out by 45°, and offset randomly by a multiple of 9°. Neutron stream intensity is also reduced by 25% to compensate for the higher output. This means neutrons are now more evenly distributed, and fixes a rare issue where by pure bad luck, all neutron streams hit the same fuel rod/reflector, causing a massive reaction spike for no good reason.
RBMK fuel rods can no longer be cycled automatically by the autoloader if the skin temperature exceeds 1,000°C, and manually removing one cannot be done unless it is colder than 200°C. Manual fuel cranes are not subject to any restrictions, and can be used to safely remove fuel when nothing else can.
RBMK columns will now undergo meltdown if broken when there's fuel loaded in them, and that fuel's skin temp exceeds 1,500°C.
The annihilator is now polluting, creating radiation when scrapping radioactive items and pollution when burning polluting fluids (with a 200% modifier).

X5617

The already deprecated T45 no longer has any resistances beyond the vanilla armor points.

X5615

There is now a new version of the industrial turbine, the old block remains with the (LEGACY) tag. The new turbine has no GUI, the steam compressor can be configured via a valve on the front. Additionally, it has a flywheel, which means it will slowly spin up when steam is supplied and spin down as steam cuts out, smoothing out the power production. The steam buffer now scales down with higher pressures, meaning that the ratio of turbines needed in a multi step setup is always 1:1:1:1.
Due to the turbine rework, the industrial turbine shares some code with the levis, which now no longer have a power buffer at all, similar to stirling engines, steam engines and MHD turbines. This removes the need for volatile power buffers which had been introduced not long ago with the battery rework.
The crucible now places materials smelted without a recipe in the recipe buffer, to allow use of materials if a recipe template is inserted after the material has been smelted. This means setups that rely on using no-recipe mode are essentially flipped now. To restore the old behavior, use the "/ntmserver set LEGACY_CRUCIBLE_RULES true" command.

X5593

Radiation absorber blocks have been changed to use metadata, the old blocks still exist with the (LEGACY) tag, however they employ a new system that automatically converts them to the new variants on chunk load.
Satellite deco blocks have been removed, they weren't exactly pretty, hideously expensive, and used up an entire block ID per piece.
Most blast doors have received new models and textures, which also changes the selection of door skins. The door skin changer item is now deprecated, skins can be cycled with the screwdriver.

X5572

Self-charging batteries have been reworked, using one item ID and meta for the new subtypes. The old items still exist with the (LEGACY) tag.

X5567

Batteries have changed fundamentally, most battery items and all existing energy storage blocks have been removed. Instead, there are new battery and capacitor items, which can be connected to a power grid using a battery socket. The socket acts similarly to the old energy storage blocks, but it has no buffer on its own without a battery item. The FEnSU now has a new version which is much larger and stores infinite energy.
Oily coal ores, legacy fusion reactor parts, the schrabidium transmutator and rare metal blocks have all been removed after being deprecated for a long time.
Steam turbines now have volatile power buffers, unused power will drain at a rate of 5% per tick.

X5544

528 mode has changed from the ground up. Coltan bedrock ore no longer spawns, instead tantalum can be obtained from bedrock ore processing.
The legacy fusion reactor core components have been removed entirely.
The ambience radiation generator and small geothermal generator have been removed. They were previously unobtainable.
Iron barrels and leaking barrels have been removed entirely. In exchange, both steel barrels and 256k tanks no longer use oil tar in their recipes, making them available early on.
Gating for the cracking tower has been adjusted, instead of polymer, it now uses desh ingots.

X5526

The multitool ("power fist") has been removed.
Explosions that used the old ZOMG projectiles (negative energy) now use a DFC blast instead.

X5523

The fusion reactor has been replaced by a new one. Existing fusion reactors will continue to work, but their components as well as their blankets are now unobtainable. Legacy reactors will drop themselves instead of their components when broken, allowing them to be moved easily. Melting reactors also no longer disassemble, instead they just break entirely.
Ammo duping recipes using nitra have been removed, instead, nitra can be used to craft variants of the ammo container which yield less, but don't require an initiall ammo item to dupe.
NITAN chests have been removed. In their place, there's something arguably more useful now.
Legacy terra drills can no longer be crafted. They were replaced by the previously added powered drill.
Helium-4 can no longer be made from liquefacting ionized particles, it is instead sourced from the fusion reactor's recipe directly.

X5490

Standard fluid identifiers which required a template folder to make and could only represent one type, have been removed. Fluid identification is now done solely though the multi identifier.

X5467

T-45 power armor has been replaced by T-51b. The recipes and stats were copied 1:1, in essence the T-51b is just a model change, however they are still separate items. Existing T-45s will continue to work for now.

X5441

Heliostat mirror mechanics have changed. The maximum distance is now much higher, but mirrors can no longer be tilted more than 45°. A larger mirror surface therefore demands the boiler to be placed further up. This angle check is done when linking, already linked mirrors (i.e. existing setups) are unaffected and should continue working as normal.

X5412

The new PUREX now handles depleted fuel recycling which was previously done by the centrifuge. The previous centrifuge recipes will no longer work, any automation setups require the centrifuges to be replaced. PUREX recipes usually also require nitric acid and kerosene in addition to the depleted fuel.

X5397

The schrabidium transmutator has been deprecated. Remaining automation options include the MEP PUREX recipes (require schrabidic acid to start) and the exposure chamber (requires PA). Existing transmutators still work, but they can no longer be made.
Assemblers, like chemical plants, have gotten a replacement. These new machines no longer require templates, they feature a built-in recipe selector. The blueprint slot is used for certain non-standard recipes that can be found in structures. Existing assemblers will continue to work, but neither them nor their templates are obtainable anymore.
