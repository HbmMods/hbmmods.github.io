Compatibility Bulletin

This list contains recent changes that need to be accounted for when updating, as well as removed features.

X5567

Batteries have changed fundamentally, most battery items and all existing energy storage blocks have been removed. Instead, there are new battery and capacitor items, which can be connected to a power grid using a battery socket. The socket acts similarly to the old energy storage blocks, but it has no buffer on its own without a battery item. The FEnSU now has a new version which is much larger and stores infinite energy.
Oily coal ores, legacy fusion reactor parts, the schrabidium transmutator and rare metal blocks have all been removed after being deprecated for a long time.
Steam turbines now have volatile power buffers, unused power will drain at a rate of 5% per tick.

X5544

528 mode has changed from the ground up. Coltan bedrock ore no longer spawns, instead tantalum can be obtained from bedrock ore processing.
The legacy fusion reactor core components have been removed entirely.
The ambience radiation generator and small geothermal generator have been removed. They were previously unobtainable.
Iron barrels and leaking barrels have been removed entirely. In exchange, both steel barrels and 256k tanks no longer use oil tar in their recipes, making them available early on.
Gating for the cracking tower has been adjusted, instead of polymer, it now uses desh ingots.

X5526

The multitool ("power fist") has been removed.
Explosions that used the old ZOMG projectiles (negative energy) now use a DFC blast instead.

X5523

The fusion reactor has been replaced by a new one. Existing fusion reactors will continue to work, but their components as well as their blankets are now unobtainable. Legacy reactors will drop themselves instead of their components when broken, allowing them to be moved easily. Melting reactors also no longer disassemble, instead they just break entirely.
Ammo duping recipes using nitra have been removed, instead, nitra can be used to craft variants of the ammo container which yield less, but don't require an initiall ammo item to dupe.
NITAN chests have been removed. In their place, there's something arguably more useful now.
Legacy terra drills can no longer be crafted. They were replaced by the previously added powered drill.
Helium-4 can no longer be made from liquefacting ionized particles, it is instead sourced from the fusion reactor's recipe directly.

X5490

Standard fluid identifiers which required a template folder to make and could only represent one type, have been removed. Fluid identification is now done solely though the multi identifier.

X5467

T-45 power armor has been replaced by T-51b. The recipes and stats were copied 1:1, in essence the T-51b is just a model change, however they are still separate items. Existing T-45s will continue to work for now.

X5441

Heliostat mirror mechanics have changed. The maximum distance is now much higher, but mirrors can no longer be tilted more than 45Â°. A larger mirror surface therefore demands the boiler to be placed further up. This angle check is done when linking, already linked mirrors (i.e. existing setups) are unaffected and should continue working as normal.

X5412

The new PUREX now handles depleted fuel recycling which was previously done by the centrifuge. The previous centrifuge recipes will no longer work, any automation setups require the centrifuges to be replaced. PUREX recipes usually also require nitric acid and kerosene in addition to the depleted fuel.

X5397

The schrabidium transmutator has been deprecated. Remaining automation options include the MEP PUREX recipes (require schrabidic acid to start) and the exposure chamber (requires PA). Existing transmutators still work, but they can no longer be made.
Assemblers, like chemical plants, have gotten a replacement. These new machines no longer require templates, they feature a built-in recipe selector. The blueprint slot is used for certain non-standard recipes that can be found in structures. Existing assemblers will continue to work, but neither them nor their templates are obtainable anymore.
